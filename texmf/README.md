## Linking my texmf files on MACOS
Creating a symbolic link that maps from SRC to DST.  This should be executed as
SUDO.  
```
ln -s "/Users/markolsonse/GitHub/mhoTexmf/texmf" /usr/local/texlive/texmf-local/tex/latex/local/"

texhash
```
